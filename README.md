# Análise de Sono em Mamíferos (Sleep Dataset)
Projeto da disciplina de Modelagem Estatística (2º Bimestre).

## Alunos
Felipe de Freitas da Silva
Benjamin Yuji Suzuki

## Objetivos
* **Regressão:** Prever tempo total de sono (`total_sleep`).
* **Classificação:** Prever perfil de sono (`sleep_class`).

## Ferramentas
* Python 3.10/11 (O PyCaret não funciona em python 3.12+)
* Bibliotecas: Pandas, Seaborn, Statsmodels, Scikit-learn, PyCaret.

## Como Rodar
1.  Faça o Download do arquivo zip ou Clone do Repositório
```bash
git clone https://github.com/ffelipef/projetoModelagem.git
```

2. Instale as dependências:
```bash
pip install -r requirements.txt
```

3. Execute o notebook `Projeto_Modelagem.ipynb`.
