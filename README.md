# Análise de Sono em Mamíferos (Sleep Dataset)
Projeto da disciplina de Modelagem Estatística (2º Bimestre).

## Objetivos
* **Regressão:** Prever tempo total de sono (`total_sleep`).
* **Classificação:** Prever perfil de sono (`sleep_class`).

## Ferramentas
* Python 3.10+
* Bibliotecas: Pandas, Seaborn, Statsmodels, Scikit-learn, PyCaret.

## Como Rodar
1. Instale as dependências: `pip install -r requirements.txt`
2. Execute o notebook `Projeto_Modelagem.ipynb`.
